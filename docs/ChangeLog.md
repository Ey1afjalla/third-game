# 版本变更日志

## [v1.0.0] - 2026-07-15

### 新增

- 完成 Stage 5 最终交付：整体美术升级、最终自检、发布分装。
- 主菜单、地下城、战斗、奖励、事件、商店、展示模式和调参面板统一为地下城战术桌面视觉。
- 随机事件选择后新增得失反馈，玩家可以确认获得/失去内容后继续。
- 商店购买和事件商人购买均会正确扣除金币。
- 发布版本号更新为 v1.0.0，并生成 Web 静态发布包。

### 改进

- 地下城地图增加符石节点、路线氛围、层级纹理和更明确的节点状态。
- 战斗界面增加阵营视觉区分、战场网格、生命条高光和日志层次。
- 奖励、商店、事件弹窗增加主题化背景、高光边线和更清晰的交互反馈。
- 主界面移除继续游戏和设置入口，保留开始冒险、快速战斗、展示模式、调参面板。
- README、QA、开发日志和任务看板同步最终版状态。
- 战斗外恢复生命效果调整为提升全队生命上限，避免开战满状态规则让恢复奖励失去价值。

### 修复

- 修复第一层战斗完成后第二层未解锁的问题。
- 修复开始冒险后点击战斗无法进入战斗的问题。
- 修复展示模式入口和主界面状态信息。
- 修复商店购买后不扣金币的问题。
- 修复随机事件中花费金币选项未实际扣金币的问题。
- 修复恢复类奖励在下一场战斗开始满状态时收益过低的问题。

### 发布

- 构建检查：`npm run build` 通过。
- 本地入口：`http://127.0.0.1:5174/`。
- 发布包：`release/mini-dungeon-commander-v1.0.0-web.zip`。
