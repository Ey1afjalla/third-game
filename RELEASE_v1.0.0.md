# Mini Dungeon Commander v1.0.0

发布日期：2026-07-15

## 版本定位

v1.0.0 是《迷你地下城指挥官》的最终 Web 发布版，完成 Stage 5：美术升级、文档同步、最终自检和发布分装。

## 主要内容

- 完整冒险流程：地下城路线、战斗、奖励、事件、商店、休息和 Boss 节点。
- 展示模式：可观察 AI 决策、状态机、行为树路径、伤害公式和战斗日志。
- 调参面板：支持玩家、敌人、战斗参数和奖励掉落的预设调整。
- 美术升级：全局地下城战术桌面主题，统一主菜单、战斗、地图、弹窗和工具界面。
- 事件反馈：随机事件选择后展示获得/失去内容。
- 经济修复：商店购买和商人事件购买会正确扣除金币。

## 下载与运行

Release 附件：

- `mini-dungeon-commander-v1.0.0-web.zip`

运行方式：

1. 解压 zip。
2. 使用静态服务器打开解压后的目录。
3. 推荐命令：

```bash
npx serve dist
```

## 验证

- `npm run build` 通过。
- `npm audit --audit-level=moderate` 通过，0 个漏洞。
- 本地浏览器验证主菜单、地下城、快速战斗界面正常。

## 已知限制

- 当前仓库没有 `src-tauri` 桌面端配置，本次提供 Web 静态发布包，不提供桌面安装包。
- macOS、Linux、Firefox、Edge 未在真实设备上完整回归。
